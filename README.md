#####################################################
#                 How to use it                     #
#####################################################
First go to chrome://extensions/
Then turn on the developer mode (in left upper corner).
Then choose Load Unpacked (in right upper corner, button).
You have to choose the folder, not the .zip file so unzip it.
Use it.
